Finish the main question bank first — these questions continue from it, so the parts here are named **D, E, F**. Every question is the same job: read the SMS, find what varies, and **write the print statement** that generated it.

Remember the pallavi: a piece is not a variable because it *looks like data* — it is a variable because it *varies*. Name every variable by what its value means — one word, capital at each joined word (`BookingDate`, never `Booking Date`). Your names may differ from the answer key's; your **strings must match exactly**.
